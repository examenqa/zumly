# Keyboard Shortcuts

A quick reference for every keyboard shortcut and mouse interaction in FollowCursor.

---

## Global Hotkeys

These work from **any application** while FollowCursor is running — you don't need to have the app window in focus.

| Shortcut | Action |
| -------- | ------ |
| **Ctrl+Shift+R** | Start or stop recording |
| **Ctrl+Shift+=** | Zoom in at the current cursor position (during recording) |
| **Ctrl+Shift+-** | Zoom out to full view (during recording) |

---

## Editor Shortcuts

These work when the FollowCursor editor window is active.

| Shortcut | Action |
| -------- | ------ |
| **Space** | Play / Pause |
| **Z** | Insert a zoom keyframe at the playhead position |
| **Delete** | Remove the selected zoom segment, video segment, voiceover segment, or click event |
| **Ctrl+Z** | Undo |
| **Ctrl+Shift+Z** or **Ctrl+Y** | Redo |
| **Ctrl+S** | Save project |

---

## Mouse Interactions

| Interaction | Where | What it does |
| ----------- | ----- | ------------ |
| Left-click | Timeline (empty) | Seek to that position |
| Left-click | Zoom segment | Select the segment |
| Left-click | Click event dot | Select the click event |
| Left-click | Video segment | Select the video segment |
| Left-click | Voiceover segment | Open the voiceover editor |
| Right-click | Preview | Add a zoom centered on that spot |
| Right-click | Preview (while zoomed in) | Add a pan path point |
| Right-click | Timeline (empty) | Add zoom or voiceover here |
| Right-click | Zoom segment | Set depth, centroid, or delete |
| Right-click | Video segment | Delete segment |
| Right-click | Pan point marker | Pick center, reorder, or delete |
| Drag edge | Zoom segment | Resize the zoom duration |
| Drag body | Zoom segment | Move the zoom to a different time |
| Drag handle | Timeline edge | Set trim start or trim end |
| Drag marker | Pan path point | Change the point's timestamp |
